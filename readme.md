# 后端本地启动
1. 项目目录下执行`pip install -r requirements.txt`
2. `Terminal(终端)`执行命令`uvicorn main:app --reload --host 0.0.0.0`
3. 访问服务
- http://127.0.0.1:8000/docs    # 接口文档
